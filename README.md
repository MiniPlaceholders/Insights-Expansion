# Insights Expansion
Insights Expansion for MiniPlaceholders

```
- <insights_limits_limit:[material|entity]>
Displays the limit name for the given material / entity type, at the current position of the player.

- <insights_limits_max:[material|entity]>
Displays the max amount allowed by the limit for the given material / entity type, at the current position of the player.

- <insights_limits_count:[material|entity]>
Displays the distribution count for the given material / entity type (includuing the rest of the limit), at the current position of the player.
```
